Networking Project

Networking / Simulation 2025

A Cisco Packet Tracer networking project that designs and simulates a complete network topology with IP addressing, routing protocols, and connectivity testing.

🎯 What I Learned

  • Designed network topologies with routers and switches
  • Applied subnetting and IP addressing schemes
  • Configured routing protocols (RIP, OSPF)
  • Troubleshot and validated connectivity

🛠 Tech Stack

  • Cisco Packet Tracer
  • Routers, Switches, PCs
  • Static & Dynamic Routing

📌 Evaluation / Next Step

The project successfully demonstrates a working network topology with correct routing and full connectivity. Limitations remain since this is a simulation; next steps involve real hardware testing, VLAN setup, security policies, and advanced protocols.
